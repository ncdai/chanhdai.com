## Features

- Automatically detects mobile device support via pointer media query.
- Uses Vibration API on Android and iOS checkbox trick on iOS.
- Gracefully degrades on desktop devices.

## API Reference

### haptic()

Triggers a brief haptic feedback (50ms vibration).

```tsx
function haptic(): void
```

Automatically detects the device platform and uses the appropriate method:

- Android/modern browsers: Vibration API
- iOS: Checkbox switch element trick

### supportsHaptic

Boolean constant that checks if the device likely supports haptic feedback.

```tsx
const supportsHaptic: boolean
```

Detects mobile devices using the `pointer: coarse` media query.
